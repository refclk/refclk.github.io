RMK Marking for Resistor

Posted on February 18, 2020

In RKM code system, the resistor’s value is marked by 3 or 4 digits and a letter L,R,K,M or G. The letter is used as a multiplier and a decimal separator.For example, 5K6 indicates a resistor value of 5.6 kΩ(5.6×1000=5.6k).
