EIA E96 Coding for Resistor

Posted on February 18, 2020

EIA E96 Coding is used for 1% precision 0603 resistors.It starts with two-digit code followed by a letter multiplier. Each letter, from “A” — “Z”, represents a specific multiplier.For example, 68X indicates a resistor value of 49.9Ω.The two-digit code refers to one of the E96 series numbers.
The standard IEC60062 gives a good explanation for it.
